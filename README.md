# Novel Structure as Code

This Python script demonstrates structuring a novel's narrative elements like characters, locations, and plot points as code. It uses classes to represent these components and a main project class to manage and 'compile' the story, showcasing a Story-Driven Development (SDD) approach for creative writing.

## Language

`python`

## How to Run

Save the code as `novel_builder.py` and run it from your terminal using: `python novel_builder.py`
